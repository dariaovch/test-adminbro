## Директории

`/public` — статика
`/models` — Схемы и модели для добавления ресурсов в БД 
`/src` — компоненты Реакт 
  
App.js конфигурирует панель администрации

## Запуск проекта

`npm run start` — запускает сервер   
`npm run dev` — запускает сервер с hot-reload
